# Curated source catalog

The guardrail against mediocre inspiration. Pull only from here (or peers at this level). **Intersect two axes:** match the *aesthetic family* to the user's stated style, and the *surface/pattern* to what they're building. This list is editable — add your own go-to sources anytime.

**Two kinds of source:**
- **Mobbin** → real shipped *in-app* screens and flows. Best for product UI: onboarding, settings, wizards, dashboards, empty states. Search by app, flow, or UI element.
- **Marketing/live sites** → public brand and landing aesthetic. Best for style tiles, landing pages, and visual direction. Visit with the browser tool.

---

## A. By aesthetic family

### Elegant / minimal / refined
The default for "clean", "modern", "premium-simple". Restraint, generous space, one confident accent.
- **Linear** (linear.app) — the reference. Marketing *and* in-app.
- **Vercel** (vercel.com), **Stripe** (stripe.com) — impeccable type, spacing, motion.
- **Mercury** (mercury.com) — refined fintech.
- **Things** (Cultured Code), **Notion**, **Notion Calendar (Cron)**, **Height** — in-app via Mobbin.

### Expressive / bold / neo-brutalist
For "expressive", "playful", "high-energy", "neo-brutalist". Heavy type, hard edges, saturated color, visible grid.
- **Gumroad** (gumroad.com) — the canonical neo-brutalist commerce brand.
- **Figma** brand/Config microsites, **Framer** (framer.com) templates, **Val Town** (val.town), **Clerk** (clerk.com) marketing.

### Premium / luxury / editorial
For "high-end", "cinematic", "editorial", "crafted". Big imagery, dramatic motion, typographic confidence.
- **Apple** (apple.com), **Teenage Engineering** (teenage.engineering), **Rivian** (rivian.com).
- **Aether**, **Igloo Inc**, **Lusion**, **Active Theory** work — award-grade WebGL/editorial (study craft, not for product UI).

### Dev-tool / technical / dense
For developer products, dashboards, technical SaaS. Dark-friendly, monospace accents, dense-but-legible.
- **Raycast** (raycast.com), **Linear**, **Railway** (railway.app), **Resend** (resend.com), **Supabase** (supabase.com), **PlanetScale**, **Neon**, **Sentry**, **Liveblocks**, **Warp**, **Arc** (arc.net).

### Fintech / trust / data-dense
For finance, crypto, data products. Trust cues, precise tables/charts, clear hierarchy.
- **Stripe**, **Mercury**, **Ramp** (ramp.com), **Brex**, **Coinbase** (coinbase.com), **Robinhood**, **Wise**, **Plaid**, **Modern Treasury**.

### Consumer / friendly / approachable
For broad-audience apps. Warmth, rounded forms, illustration, friendly voice.
- **Airbnb** (airbnb.com), **Duolingo**, **Headspace**, **Calm**, **Pitch** (pitch.com), **Cash App**, **Spotify**, **Revolut**.

### Playful / characterful
For personality-forward products. Mascots, motion, delight.
- **Duolingo**, **Gumroad**, **Arc**, **Discord**, **Headspace**, **Poppy**.

## C. Denylist — never use

These are not production-grade truth; skip them entirely:
- **Dribbble, Behance, Pinterest** — concept shots, not shipped or pressure-tested.
- **Template marketplaces** (ThemeForest, etc.) and **AI-image/UI galleries**.
- **"Top 10 UI" listicles** and unsourced aggregators.

When tempted, go to the **actual product** instead — the live site or its real screens on Mobbin.
